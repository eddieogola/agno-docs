It provides the complete stack for building, running and managing multi-agent systems.

| Layer | What it does |
|-------|--------------|
| **SDK** | The primitives: agents, teams, workflows, memory, knowledge, guardrails, and governance. |
| **Engine** | The agent loop: model calls, tool execution, context management, memory retrieval. |
| **AgentOS** | The runtime: serve your agents as production software. Streaming, authentication, request isolation, approval enforcement, and a [control plane](https://os.agno.com) to monitor and manage everything. |

## Why Agno?

Agentic software operates under a different contract than traditional software.

**A new interaction model.** Traditional software accepts requests and returns responses. Agents stream reasoning, tool calls, and results in real time. An agent can start reasoning, retrieve new information from memory, and pivot entirely. It can pause mid-execution, wait for input or approval, and resume days later. Agents require a fundamentally different interaction model and Agno supports it natively.

**A new governance model.** Traditional software executes instructions. There's nothing to decide, so there's nothing to govern. Agents make decisions, and not all decisions are equal. Some decisions can be made freely, some need user approval, and some need admin approval. Agno lets you express who decides what as part of the agent definition, with approval workflows, human-in-the-loop, and audit logs built in.

**A new trust model.** Traditional software is predictable. Same input, same output, every time. Agents aren't. You can't trust the output the way you trust a database query. So the system has to monitor itself. Agno runs guardrails and evaluations as part of the engine, not as a separate tool you bolt on.

Agno is built to natively express the new contract of agentic software.

## Quick Example

An agent with MCP tools, persistent state, and a production-ready API:

```python
from agno.agent import Agent
from agno.db.sqlite import SqliteDb
from agno.models.anthropic import Claude
from agno.os import AgentOS
from agno.tools.mcp import MCPTools

agent = Agent(
    name="Agno Agent",
    model=Claude(id="claude-sonnet-4-5"),
    db=SqliteDb(db_file="agno.db"),
    tools=[MCPTools(url="https://docs.agno.com/mcp")],
    add_history_to_context=True,
    markdown=True,
)

agent_os = AgentOS(agents=[agent])
app = agent_os.get_app()

if __name__ == "__main__":
    agent_os.serve(app="agno_agent:app", reload=True)
```
